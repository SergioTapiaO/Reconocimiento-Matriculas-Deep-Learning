Trabajo de fin de grado - Ingenieria del software UCM
Reconocimiento de matrículas de vehículos en imágenes mediante técnicas de Aprendizaje Profundo aplicado a TIC
Nota obtenida: 10

El proyecto consta de varias partes: 
I. Por un lado hemos entrenado las redes AlexNet y GoogleNet mediante recortes de fotos de vehiculos para que identificara la matrícula del vehículo a partir de una imagen (Matlab).

II. Una vez obtenemos el recorte con la matricula, utilizamos un OCR para que identifique las letras y numeros de la matricula (Matlab).

III. Desarrollamos una aplicacion web que funciona a modo de parking comunitario, donde cada vecino tiene registrados sus coches, junto con la informacion de cual esta dentro del parking y si esta dentro del parking (JavaScript, NodeJS, html y css.

IV. Una vez esta listo todo lo anterior, desarrollamos con Matlab un programa para simular la entrada o salida de un coche al parking, el programa solicita una foto del coche, a partir de esta foto se utiliza el aplicativo descrito en los puntos I y II para obtener la matricula del vehiculo, y una vez tenemos la matricula se envia una peticion a la API de la aplicacion web con dicho identificador para que registre el movimiento en el parking.
